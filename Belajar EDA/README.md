# Note

Nah EDA ini singkatnya tempat kita memahami untuk Data, EDA dilakukan pertama kali setelah kita melakukan split pada data yang tujuannya kita tidak atau jangan melihat data yang dijadikan sebagai data testing. Nah EDA ini juga disebut sebagai pre-model insight yang maksudnya ketika kita melakukan EDA disini juga kita harus Memahami Model yang akan digunakan / ditest nanti sebagai perbandingan, setiap model yang digunakan memiliki tambahan-tambahan proses EDA yang berbeda, tetapi pada dasarnya tetap sama yaitu Univariate Analysis, Bivariate Analysis, dan Multivariate Analysis (Saya tidak menemukan bagian ini).

Untuk bagian dasar ini akan saya ambil dari github adilshamim8, dan bagian EDA yang diperlukan setiap model algoritma memerlukan tahap lebih lanjut terkait dengan pemahaman teknis model itu sendiri seperti Linear atau Non-linearnya model, mungkin kalau sudah selesai saya akan menambahkan bagian dasar dasar yang telah saya sebutkan khususnya bagian implementasinya karena saya juga tertarik bagian data visualization nya dan juga saya akan menambahkan folder tingkat lanjut untuk EDA nya dan model apa saja yang memerlukan EDA seperti ini.
