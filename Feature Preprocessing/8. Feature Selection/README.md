# Note
Ketika kita melakukan training pada model kita tidak bisa untuk memasukkan semua feature yang ada pada dataset, efeknya bisa membuat model Overfitting, kita juga bisa melakukan performa cek dengan learning curve, ada berbagai kondisi yang bisa diperhatikan untuk melihat langkah langkah selanjutnya seperti underfitting atau overfitting, kita juga harus memperhatikan ident feature dan juga nilai kardinalitas yang bisa terlalu rendah atau terlalu tinggi yang ada pada dataset seperti passengger id atau pun nama seseorang yang tidak memiliki pola untuk model belajar. 

Untuk modul preprocessing bagian "Feature Selection" akan saya tambahkan dari github adilshamim8 dan menambahkan beberapa teknik advance seperti untuk data linear seperti Forward Selection dan Backward Selection.
