# Roadmap Feature Preprocessing untuk Machine Learning

## 1. Data Cleaning
Membersihkan data yang tidak valid atau bermasalah.

### Materi
- Missing Values
- Duplicate Data
- Inconsistent Data
- Data Type Conversion
- Handling Invalid Values

---

## 2. Missing Value Handling
Menangani nilai yang hilang.

### Materi
- Mean Imputation
- Median Imputation
- Mode Imputation
- Constant Value Imputation
- KNN Imputation
- Iterative Imputation

---

## 3. Outlier Handling
Menangani data ekstrem.

### Materi
- IQR Method
- Z-Score
- Winsorization
- Capping
- Removal Outlier

---

## 4. Encoding Categorical Features
Mengubah data kategorikal menjadi numerik.

### Materi
- Label Encoding
- Ordinal Encoding
- One Hot Encoding
- Target Encoding
- Frequency Encoding

---

## 5. Feature Scaling
Menyamakan skala antar fitur.

### Materi
- StandardScaler
- MinMaxScaler
- RobustScaler
- MaxAbsScaler

---

## 6. Feature Transformation
Mengubah distribusi data.

### Materi
- Log Transformation
- Square Root Transformation
- Box-Cox Transformation
- Yeo-Johnson Transformation

---

## 7. Feature Construction
Membuat fitur baru dari fitur yang ada.

---

## 8. Feature Selection
Memilih fitur terbaik.

### Filter Method
- Correlation
- Chi-Square
- ANOVA

### Wrapper Method
- Forward Selection
- Backward Elimination
- RFE

### Embedded Method
- Lasso
- Random Forest Feature Importance
- XGBoost Importance

---

## 9. Multicollinearity Handling
Mengatasi fitur yang terlalu berkorelasi.

### Materi
- Correlation Matrix
- Variance Inflation Factor (VIF)

---

## 10. Imbalanced Data Handling
Menangani distribusi kelas yang tidak seimbang.

### Materi
- Random Oversampling
- Random Undersampling
- SMOTE
- ADASYN

---

## 11. Pipeline dan Automation
Mengotomatisasi preprocessing.

### Materi
- Pipeline
- ColumnTransformer

- Cross Validation Pipeline

---

## 12. Advanced Feature Engineering

### Materi

- Binning
- Clustering Features

## 13. Khusus Data Waktu (Time Series)

### Materi

## 14. Khusus Data Teks (NLP)
